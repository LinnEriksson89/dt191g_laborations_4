# Uppgift 4 - ASP.NET Core API

Skapa ett API med ASP.NET Core. APIet ska ha fullt stöd för CRUD. 

APIet ska innehålla valfri data, minst tre olika fält. APIet ska gå att använda med ett testverktyg för REST APIer.

Det är valfritt att göra ett fullständigt API med controller eller ett minimalt API.
